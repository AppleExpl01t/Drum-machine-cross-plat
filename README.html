‚ö° Getting Started

1. Installation

Open your terminal in the project root and run:

npm install

2. Development Mode

To start the app on your desktop:

npm run tauri dev

3. Android Dev

Connect your Android device (USB Debugging ON) or start Emulator:

npm run tauri android dev
